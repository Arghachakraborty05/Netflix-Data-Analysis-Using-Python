# Netflix-Data-Analysis-Using-Python

## Project Overview
This project focuses on **analyzing Netflix movies and TV shows data** to uncover meaningful insights related to content distribution, trends, and audience preferences.  
The analysis was performed using **Python in Google Colab**, making use of data manipulation and visualization techniques to support **business and analytical decision-making**.

---

## Objectives
- Understand the distribution of Movies and TV Shows on Netflix  
- Analyze content trends over the years  
- Identify popular genres and countries producing Netflix content  
- Gain insights that can help in **content strategy and business decisions**

---

## Dataset Description
- Dataset contains information about Netflix titles such as:
  - Title  
  - Type (Movie / TV Show)  
  - Country  
  - Release Year  
  - Genre  
  - Duration  
  - Rating  
- The dataset is publicly available and widely used for exploratory data analysis.
